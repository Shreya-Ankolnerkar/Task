# ConnectClient

## Description  
A full-stack web application featuring a modern landing page and an admin panel. The landing page displays projects dynamically fetched from the backend, along with client information, contact details, and newsletter subscriptions. The admin panel allows administrators to manage projects and clients, view contact form submissions, and monitor subscriber data with an intuitive, clean interface.

## Features / What it does  
1. Landing Page Features

Hero Section: A clean introduction banner for your application.

Our Projects Section:

Fetch and display all projects from the backend.

Each project shows an image, name, description, and a non-functional “Read More” button.

Clients Section: Displays client/company logos or details.

Contact Section (Separate Page): Users can submit inquiries, stored in the backend.

Newsletter Subscription (Separate Page): Users can subscribe with their email; data sent to backend. 


2. Admin Panel Features

Add Project

Add new projects with name, manager, description, and dates.

Add Client

Add client/company details and logos.

View Contact Form Submissions

Admin can see all contact requests sent by users.

View Newsletter Subscribers

Admin can track and manage subscriber list.

Responsive Navigation

Smooth navigation between admin dashboard sections.
 

## Tech Stack / Technologies Used  
Java, Spring Boot, MySQL, React, HTML, CSS, JavaScript etc.
